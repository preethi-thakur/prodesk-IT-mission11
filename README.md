# 🎬 Movie Explorer

A responsive Movie Explorer web application built with **Next.js**, **React**, and **Redux Toolkit**. The application allows users to browse and search movies through an intuitive interface while demonstrating component-based architecture, state management, and unit testing.

## 🚀 Features

- Search and explore movies
- Responsive user interface
- Reusable React components
- State management with Redux Toolkit
- Fast rendering with Next.js
- Unit testing using Jest and React Testing Library
- Clean and maintainable project structure

## 🛠️ Tech Stack

- Next.js 15
- React 19
- Redux Toolkit
- React Redux
- Jest
- React Testing Library
- JavaScript (ES6+)

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Move into the project directory:

```bash
cd movieexplorer
```

Install dependencies:

```bash
npm install
```

## ▶️ Run the Development Server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

## 🧪 Run Tests

```bash
npm test
```

Watch mode:

```bash
npm run test:watch
```

## 🏗️ Build for Production

```bash
npm run build
```

Start the production server:

```bash
npm start
```
